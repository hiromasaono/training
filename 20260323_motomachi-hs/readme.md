## 今日の探偵ミッション
卵アレルギーの人の中には、不思議なことに：
- ニワトリの卵 → アレルギー反応が出る😣
- ウズラの卵 → 大丈夫な人もいる😊

**なぜ？** この謎を解くため：
1. オボムコイドという卵の中のタンパク質を調べる ([オボムコイドって何?](https://www.pt-bio.com/project/egg-for-all))
   -  ⚠️ URLリンクは右クリックメニューの「新しいタブで開く」とよいです
2. 様々な生物の似たタンパク質を比べる
3. その違いとアレルギーの関係を探る

## どうやって調べるの？
1. 世界中の研究者が使うデータベースでアレルギーに関係するタンパク質の配列を探す
2. タンパク質配列を比較して違いを見つける
3. 【発展編】分子系統樹を作って、生物間の関係を調べる
 
- これらの作業を通じて、「どの生物の卵がアレルギーを起こしやすいか」を予測する手がかりを得ることができます！

   ### 遺伝子とタンパク質の関係
想像してみてください：

```
     📝      →      🏗️
```

## 🔰 Step 1　調査手順

### 1️⃣ 基本の検索
1. [UniProt](https://www.uniprot.org/)のウェブサイトにアクセス
2. 中央の検索窓に「**ovomucoid**」と入力
   - ⚠️ 英語で入力しましょう！
   - 「Table」表示が選択されていることを確認
3. オボムコイドを持つ生物が[リスト](https://www.uniprot.org/uniprotkb?query=ovomucoid)になって表示されます。
![](https://raw.githubusercontent.com/hiromasaono/training/master/images/20250121_preexpo_01.jpg)



### 2️⃣ 検索結果の見方
検索結果には以下の情報が表示されます：
- **Entry**: タンパク質のID（例：P01005）
- **Entry Name**: タンパク質の略称と生物名
- **Protein Names**: タンパク質の正式名称
- **Organism**: 生物の種名（学名と英名）
- **Length**: アミノ酸の数

### 3️⃣ BLAST検索で似たタンパク質を探す
> 🔍 **BLAST(ぶらすと)とは?**
> - BLASTは「Basic Local Alignment Search Tool（ベーシック ローカル アラインメント サーチ ツール）」の略で、生物学者の「万能検索ツール」です！	
> - タンパク質や遺伝子の「似ているものを探す」ことができます。
> - 検索結果の読み方
>   - 上位にあるほど、よく似たタンパク質
>   - スコアが高いほど類似度が高い
>   - 生物種名から進化的な関係も分かる！
1. ニワトリ(Gallus gallus)のオボムコイド([P01005](https://www.uniprot.org/uniprotkb/P01005/entry))をクリック
2. 画面中央メニューの「Tools」から「BLAST」をクリック
![](https://raw.githubusercontent.com/hiromasaono/training/master/images/20250121_preexpo_02.jpg)
3. 画面下部の「Run BLAST」で検索開始
![](https://raw.githubusercontent.com/hiromasaono/training/master/images/20250121_preexpo_12.jpg)
4. 2分ほど待って結果を確認
![](https://raw.githubusercontent.com/hiromasaono/training/master/images/20250121_preexpo_03.jpg)
    - [時間がかかるとき用のリンク](https://www.uniprot.org/blast/uniprotkb/ncbiblast-R20250212-082523-0510-69392180-p1m/overview)（⚠️ 過去の検索結果のため、期限切れでアクセスできない場合があります。その場合は自分で実行した結果を使いましょう）
5. 左側の`Status`から`Reviewed(Swiss-Prot)`をクリックすると高品質なものだけに絞り込めます
![](https://raw.githubusercontent.com/hiromasaono/training/master/images/20250121_preexpo_04.jpg)

## 🔄 Step 2: 類似する複数のタンパク質配列を並べて比較してみよう！

### 🎯 このステップのゴール
- 配列アラインメントの基本を理解する
- タンパク質の進化の痕跡を見つける
- 配列の違いからアレルギーの謎に迫る

### 💡 配列アラインメントって何？
配列アラインメントとは類似した配列を横に並べて比較する方法のことで、まるで「タンパク質のパズル」です！

### 1️⃣ 基本的な考え方
```
例えば、こんな単語のパズル：
FIGHTERS  →  FIGHTERS
FEWERS    →  F-EW-ERS
              ↑   ↑
            同じ文字を
            合わせる！
```
<details>
  
<summary> 🔍️ もっとくわしく知りたい! 🔎</summary> 

### 2️⃣ 実際のアミノ酸配列での例
```
ニワトリ: MKWVTFIS--LLLFS
ウズラ　: MKWVTFISLLLLLFS
アヒル　: MKWATFIS--LLLFT
          ***.**** ****.*
          │  │  │  │  │
          └──┴──┴──┴──┴── *マークは同じアミノ酸！
```

### 3️⃣ 配列アラインメントで分かること
1. **保存された部分** (完全に一致)
   - タンパク質の重要な機能を担う可能性が高い
   - 進化の過程で変化しにくい部分

2. **似ているけど少し違う部分** (似たアミノ酸に置換)
   - 機能は似ているが少し異なるかも
   - 生物種による特徴の違いを生む可能性

3. **大きく異なる部分** (全く違うアミノ酸)
   - 新しい機能を獲得した可能性
   - アレルギー反応の違いに関係するかも！

4. **ギャップ** (-で示される部分)
   - アミノ酸の挿入や欠失が起きた
   - 進化の過程で大きな変化があった証拠

### 4️⃣ なぜ配列アラインメントするの？
1. **機能の予測**
   - 似ている配列は似た機能を持つ可能性
   - 未知のタンパク質の役割を推測できる

2. **進化の研究**
   - どの部分が重要で保存されているか
   - どのように変化してきたか

3. **アレルギー研究への応用**
   - アレルギーを引き起こす部分の特定
   - 新しい食品のアレルギーリスクの予測


> 💡 **研究のヒント**  
> アラインメントを読むときは、まず「*」がたくさん並んでいる領域を探してみましょう。
> そこが重要な機能を持つ可能性が高い領域です！

#### 2. 特殊なアミノ酸と修飾アミノ酸
- システイン二量体 (Cys)₂
- 4-ヒドロキシプロリン (4Hyp)
- ヒドロキシリシン (Hyl)

#### 3. 重要な補足
1. Asx は Asp または Asn を表す（一文字記号: B）
2. Glx は Glu または Gln を表す（一文字記号: Z）

#### 4. アミノ酸の性質による分類
- **酸性アミノ酸**: アスパラギン酸、グルタミン酸
- **塩基性アミノ酸**: ヒスチジン以下（ヒスチジン、リシン、アルギニン）
- **中性アミノ酸**: 上記以外のすべて

> 💡 **研究のコツ**  
> アミノ酸配列を読むときは、一文字記号を使うと効率的です。20種類の基本的なアミノ酸をこの記号で覚えることで、配列解析がスムーズになります。

### 🔍 見方のポイント：

1. **配列ID**
   - 左端に表示される`P01005`などの文字列
   - UniProtのエントリー番号を示しています

2. **アミノ酸配列**
   - 一文字のアルファベットで表示
   - 例：`M`（メチオニン）、`K`（リシン）など

3. **比較のマーク**
   - `*`：完全に一致している部位
   - `:`：化学的性質が似ているアミノ酸
   - `.`：やや似ているアミノ酸
   - 空白：全く異なるアミノ酸
   - `-`：ギャップ（挿入や欠失）

4. **数字**
   - 右端の数字は、そこまでのアミノ酸の数を示します

### 💡 アレルギーとの関係を探るポイント

1. **システイン（C）の位置**
   - システインは立体構造の形成に重要
   - アレルギー反応に関わる可能性が高い
   - 種間で保存されているか確認！

2. **変異の多い領域**
   - `*`が少ない部分に注目
   - アレルギーの強さの違いに関係？

3. **N末端シグナル配列**
   - 配列の最初の部分（20-30アミノ酸）
   - 分泌に関わる部分なので、違いがあるかも

4. **注目すべきアミノ酸**
   - システイン（C）：構造維持に重要
   - プロリン（P）：構造の曲がり目
   - 荷電アミノ酸（D, E, K, R）：表面にあることが多い

#### 📋 まず、アラインメント結果を観察しよう

BLAST検索の結果画面を見ながら、以下のポイントを確認してみましょう：

- **アミノ酸の総数（Length）**: 各鳥のオボムコイドは何アミノ酸？ 長さに違いはある？
- **ニワトリとの一致率（Identity %）**: BLAST結果に表示されている数値を比べてみよう。どの鳥が最もニワトリに似ている？
- **ギャップ（`-`）の有無**: アラインメント中に `-` が入っている箇所はどこ？ それはどの鳥？

> 💡 一致率（Identity）の数値が大きいほどニワトリのオボムコイドに「似ている」ことを意味します。口頭で共有し合いながら比べてみましょう。

#### 🔬 テーマ1: なぜウズラの卵は食べられる人がいる？

- アラインメントの中で、ニワトリとウズラで **異なるアミノ酸** の箇所を数えてみよう
- その違いは配列のどのあたりに集中している？（前半？後半？散在？）
- 👥 **発表**: 各自が見つけた「ここが違う！」というポイントを1つずつ挙げてみよう

<summary>💡 ヒント（クリックして開く）</summary>

- オボムコイドは3つの「ドメイン」（機能単位）で構成されています。アラインメント結果を3つのブロックに分けて見ると、ドメインごとの違いが見えてきます
- 特に **第3ドメイン**（配列の後半部分）がアレルギー反応に重要だとされています
- ニワトリとウズラの第3ドメインにはどのくらい違いがある？

</details>

#### 🔬 テーマ2: アミノ酸の違いは何を意味する？

単に「違う」だけでなく、**どう違うか** が重要です：
- 性質の似たアミノ酸への置換（例：ロイシン L → イソロイシン I）→ 影響は小さい？
- 性質の大きく異なる置換（例：荷電アミノ酸 → 非荷電アミノ酸）→ 影響が大きい？
- 👥 **議論**: アラインメントの記号（`*` `:` `.` 空白）を手がかりに、「影響が大きそうな違い」と「影響が小さそうな違い」を分類してみよう

<details>

<summary>💡 ヒント（クリックして開く）</summary>

- アレルゲンが免疫システムに認識されるとき、タンパク質の **表面** にあるアミノ酸が重要
- 荷電アミノ酸（D, E, K, R）や親水性アミノ酸は表面に露出しやすい
- これらのアミノ酸が別の性質のアミノ酸に変わっていたら、免疫システムの認識が変わるかも？

</details>

#### 🔬 テーマ3: 進化とアレルギーの関係

- 4種の鳥の中で、ニワトリに最も配列が似ているのはどれ？最も違うのは？
- 配列の類似度と、鳥の進化的な近さには関係がありそう？
- 👥 **予測**: 「配列の似ている鳥の卵は、ニワトリ卵にアレルギーのある人にとっても危険」と言えるだろうか？ その根拠は？

<details>

<summary>💡 ヒント（クリックして開く）</summary>

- ニワトリ・ウズラ・七面鳥はキジ目（Galliformes）の仲間です
- エミューはダチョウ目（Struthioniformes）で、かなり遠い関係です
- 進化的に近い鳥ほどオボムコイドの配列が似ている傾向があります。実際のアラインメント結果と一致していますか？
- **交差反応性**：似たタンパク質が同じ抗体に認識される現象のこと。配列の類似度が高いほど交差反応が起こりやすいと考えられます

</details>

### 🌳 分子系統樹の世界へようこそ！

#### 系統樹って何？
系統樹は、生物の「家系図」のようなものです。生物がどのように進化して今の形になったのか、その歴史を木の形で表現します。

```
         ヒト 🧍
        ╱
猿 🐒──┤
        ╲
         チンパンジー 🦍

このように、枝分かれで関係性を表現！
```

#### 分子系統樹のすごいところ
1. **進化の道筋を可視化**
   - どの生物が近い関係なのか一目で分かる
   - いつ頃分かれたのかも推測できる

2. **現代の応用例：新型コロナウイルス**
   - デルタ株やオミクロン株の関係を理解
   - ウイルスの進化を追跡
   - 新しい変異株の監視に活用

3. **遺伝子の歴史を探る**
   - 遺伝子がいつ失われたか
   - 新しい機能はいつ獲得されたか
   - 似た配列の機能を予測

### 🧪 実践！分子系統樹で分かること

