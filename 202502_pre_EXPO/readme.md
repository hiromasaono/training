## 🔄 Step 2: 類似する複数のタンパク質配列を並べて比較しよう！

### 🎯 このステップのゴール
- 配列アラインメントの基本を理解する
- タンパク質の進化の痕跡を見つける
- 配列の違いからアレルギーの謎に迫る

#### 💡 配列アラインメントって何？
配列アラインメントは、類似した配列を横に並べて比較する作業のことで、まるで「タンパク質のパズル」です！

#### 1️⃣ 基本的な考え方
```
例えば、こんな単語のパズル：
FIGHTERS  →  FIGHTERS
FEWERS    →  F-EW-ERS
              ↑   ↑
            同じ文字を
            合わせる！
```
<details>
  
<summary> 🔍️ もっとくわしく知りたい! 🔎</summary> 

#### 2️⃣ 実際のアミノ酸配列での例
```
ニワトリ: MKWVTFIS--LLLFS
ウズラ　: MKWVTFISLLLLLFS
アヒル　: MKWATFIS--LLLFT
          ***.**** ****.*
          │  │  │  │  │
          └──┴──┴──┴──┴── *マークは同じアミノ酸！
```

#### 3️⃣ 配列アラインメントで分かること
1. **保存された部分** (完全に一致)
   - タンパク質の重要な機能を担う可能性が高い
   - 進化の過程で変化しにくい部分

2. **似ているけど少し違う部分** (似たアミノ酸に置換)
   - 機能は似ているが少し異なるかも
   - 生物種による特徴の違いを生む可能性

3. **大きく異なる部分** (全く違うアミノ酸)
   - 新しい機能を獲得した可能性
   - アレルギー反応の違いに関係するかも！

4. **ギャップ** (-で示される部分)
   - アミノ酸の挿入や欠失が起きた
   - 進化の過程で大きな変化があった証拠

#### 4️⃣ なぜ配列アラインメントするの？
1. **機能の予測**
   - 似ている配列は似た機能を持つ可能性
   - 未知のタンパク質の役割を推測できる

2. **進化の研究**
   - どの部分が重要で保存されているか
   - どのように変化してきたか

3. **アレルギー研究への応用**
   - アレルギーを引き起こす部分の特定
   - 新しい食品のアレルギーリスクの予測

#### 5️⃣ 配列アラインメントの読み方のコツ
```
記号の意味：
* = 完全一致（重要！）
: = 似たアミノ酸（まあまあ重要）
. = やや異なる（あまり重要でない）
- = 挿入/欠失（大きな変化）

例：
MKWVTFIS*LLLFS  ← 元の配列
MKW-TFIS*LTLFS  ← 比較する配列
***.**** *.*.*  ← 比較結果
```

> 💡 **研究のヒント**  
> アラインメントを読むときは、まず「*」がたくさん並んでいる領域を探してみましょう。
> そこが重要な機能を持つ可能性が高い領域です！

## 🎓 もっと深く知りたい人へ

### 🌳 分子系統樹の世界へようこそ！

#### 系統樹って何？
系統樹は、生物の「家系図」のようなものです。生物がどのように進化して今の形になったのか、その歴史を木の形で表現します。

```
         ヒト 🧍
        ╱
猿 🐒──┤
        ╲
         チンパンジー 🦍

このように、枝分かれで関係性を表現！
```

#### 分子系統樹のすごいところ
1. **進化の道筋を可視化**
   - どの生物が近い関係なのか一目で分かる
   - いつ頃分かれたのかも推測できる

2. **現代の応用例：新型コロナウイルス**
   - デルタ株やオミクロン株の関係を理解
   - ウイルスの進化を追跡
   - 新しい変異株の監視に活用

3. **遺伝子の歴史を探る**
   - 遺伝子がいつ失われたか
   - 新しい機能はいつ獲得されたか
   - 似た配列の機能を予測

### 🧪 実践！分子系統樹で分かること

#### アレルギータンパク質の例
```
ニワトリ
   ├── ウズラ
   │    └── 七面鳥
   └── アヒル
        └── ダチョウ

このように並ぶと...
近い関係の鳥ほど、アレルギー反応も似ている！？
```

#### 遺伝子の運命を追跡
例：ビタミンC合成遺伝子の物語
- 多くの動物：🟢 持っている
- ヒトを含む霊長類：❌ 失った
- いつ失った？→系統樹で分かる！

### 🔬 研究の最前線

1. **新薬開発**
