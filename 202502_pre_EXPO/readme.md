### 💡 UniProt(ゆにぷろっと)って何？
- 世界中の研究者が使う**タンパク質のビッグデータベース**
- なんと約2億5000万個ものタンパク質の情報が登録されている！
- ヒトから細菌まで、様々な生物のデータが無料で見られる

### 🔰 調査手順

#### 1️⃣ 基本の検索
1. [UniProt](https://www.uniprot.org/)のウェブサイトにアクセス
2. 中央の検索窓に「**ovomucoid**」と入力
   - ⚠️ 英語で入力しましょう！
   - 「Table」表示が選択されていることを確認
3. オボムコイドを持つ生物が[リスト](https://www.uniprot.org/uniprotkb?query=ovomucoid)になって表示されます。
![](https://raw.githubusercontent.com/hiromasaono/training/master/images/20250121_preexpo_01.jpg)



#### 2️⃣ 検索結果の見方
検索結果には以下の情報が表示されます：
- **Entry**: タンパク質のID（例：P01005）
- **Entry Name**: タンパク質の略称と生物名
- **Protein Names**: タンパク質の正式名称
- **Organism**: 生物の種名（学名と英名）
- **Length**: アミノ酸の数

#### 3️⃣ BLAST検索で似たタンパク質を探す
> 🔍 **BLAST(ぶらすと)とは?**
> - BLASTは「Basic Local Alignment Search Tool（ベーシック ローカル アラインメント サーチ ツール）」の略で、生物学者の「万能検索ツール」です！	
> - タンパク質や遺伝子の「似ているものを探す」ことができます。
> - 検索結果の読み方
>   - 上位にあるほど、よく似たタンパク質
>   - スコアが高いほど類似度が高い
>   - 生物種名から進化的な関係も分かる！
1. ニワトリ(Gallus gallus)のオボムコイド([P01005](https://www.uniprot.org/uniprotkb/P01005/entry))をクリック
2. 画面中央メニューの「Tools」から「BLAST」をクリック
![](https://raw.githubusercontent.com/hiromasaono/training/master/images/20250121_preexpo_02.jpg)
3. 画面下部の「Run BLAST」で検索開始
4. 2分ほど待って結果を確認
![](https://raw.githubusercontent.com/hiromasaono/training/master/images/20250121_preexpo_03.jpg)
5. 左側の`Status`から`Reviewed(Swiss-Prot)`をクリックすると高品質なものだけに絞り込めます
![](https://raw.githubusercontent.com/hiromasaono/training/master/images/20250121_preexpo_04.jpg)

## 🔄 Step 2: タンパク質配列を並べて謎を解こう！

### 🎯 このステップのゴール
- 配列アラインメント（配列並べ）の基本を理解する
- タンパク質の進化の痕跡を見つける
- 配列の違いからアレルギーの謎に迫る

### 💡 アラインメントって何？
配列アラインメントは、まるで「タンパク質のパズル」です！
```
例えば、こんな感じ：
FIGHTERS  →  FIGHTERS
FEWERS    →  F-EW-ERS
              ↑   ↑
            同じ文字を
            合わせる！
```

### 🧩 アラインメントで分かること
1. **保存された部分**（よく似ている箇所）
   - タンパク質の重要な機能を担っている可能性が高い
   - 進化の過程で変わりにくかった部分

2. **変化した部分**
   - 種によって異なる働きをする可能性
   - アレルギーの強さの違いに関係するかも？

### 🔬 実験手順

#### 1️⃣ 準備
1. 配列データファイル（multi_tax.fa）をダウンロード
   - 魚類から哺乳類まで、様々な生物の配列が含まれています
   
2. [Clustal Omega](https://www.ebi.ac.uk/Tools/msa/clustalo/)にアクセス
   - 世界中の研究者が使う配列並べプログラム

#### 2️⃣ アラインメントの実行
1. "Choose File"をクリック
2. ダウンロードしたファイルを選択
3. "Submit"をクリック
4. 結果が表示されるまで約2分待つ

#### 3️⃣ 結果の見方
```
配列の例：
MKVVLVL--LAVALVGASP  (ニワトリ)
MKVVLVLLTLAVALVGASP  (ウズラ)
MKVVLAL--LAVALVGTSP  (アヒル)
**************** **   ← 記号の意味は下記
```

**記号の意味**
- `*` : 全種で同じアミノ酸（完全に保存）
- `:` : 似たアミノ酸に置き換わっている
- `.` : やや異なるアミノ酸
- `-` : アミノ酸の挿入や欠失

> 💡 **研究者のコツ**
> - アミノ酸は20種類あり、1文字で表されます
> - 保存された領域（*が多い部分）は重要な機能を持つ可能性が高い
> - 種間で違いがある部分は、機能の違いを生む可能性があります

## 🎓 もっと深く知りたい人へ

### 🌳 分子系統樹の世界へようこそ！

#### 系統樹って何？
系統樹は、生物の「家系図」のようなものです。生物がどのように進化して今の形になったのか、その歴史を木の形で表現します。

```
         ヒト 🧍
        ╱
猿 🐒──┤
        ╲
         チンパンジー 🦍

このように、枝分かれで関係性を表現！
```

#### 分子系統樹のすごいところ
1. **進化の道筋を可視化**
   - どの生物が近い関係なのか一目で分かる
   - いつ頃分かれたのかも推測できる

2. **現代の応用例：新型コロナウイルス**
   - デルタ株やオミクロン株の関係を理解
   - ウイルスの進化を追跡
   - 新しい変異株の監視に活用

3. **遺伝子の歴史を探る**
   - 遺伝子がいつ失われたか
   - 新しい機能はいつ獲得されたか
   - 似た配列の機能を予測

### 🧪 実践！分子系統樹で分かること

#### アレルギータンパク質の例
```
ニワトリ
   ├── ウズラ
   │    └── 七面鳥
   └── アヒル
        └── ダチョウ

このように並ぶと...
近い関係の鳥ほど、アレルギー反応も似ている！？
```

#### 遺伝子の運命を追跡
例：ビタミンC合成遺伝子の物語
- 多くの動物：🟢 持っている
- ヒトを含む霊長類：❌ 失った
- いつ失った？→系統樹で分かる！

### 🔬 研究の最前線

#### 分子系統樹の活用例
1. **新薬開発**
   - 似たタンパク質の機能予測
   - 副作用の可能性の予測

2. **生物の保護**
   - 絶滅危惧種の関係性理解
   - 保護戦略の立案

3. **食品安全**
   - アレルギー反応の予測
   - 新しい食材の安全性評価
